Testimonials

“ADS CFD is pleased to feature our Code Leo flow solver in the initiative by Microsoft and Capgemini. This program reaffirms the GPU acceleration capabilities of Code Leo, which can be deployed on Azure HPC platform. This translates to the simulation of next generation designs using larger geometric complexity with higher fidelity analysis without impacting the design cycle time.” - Michael Ni, Product Manager, AeroDynamic Solutions

“Altair is pleased to participate in this program with Microsoft and Cap Gemini. Having our solvers validated on the Azure infrastructure saves time and resources for Altair and provides an objective endorsement for our mutual customers running Altair workloads on Azure.” - Mike Pautz, Director Strategic Alliances, Altair

“Pleased to participate in the validation effort by Microsoft and Capgemini for VRED deployment on Microsoft Azure. The featured report here re-confirms the GPU rendering performance of VRED. Our clients can now carry out heavy rendering computation using VRED, on Azure HPC cloud platform” - Lukas Fäth, Senior Product Manager - Automotive Visualization at Autodesk

“CPFD Software is pleased to validate Barracuda Virtual Reactor deployment on the Azure HPC Platform in this program with Microsoft and Capgemini. Azure's multi-GPU virtual machines powered by NVIDIA A100 Tensor Core GPUs provide the global Barracuda Virtual Reactor user community with instant access to the latest high performance computing resources without the overhead of purchasing and maintaining on-premise hardware. The independent benchmarking provided by Capgemini gives our clients verified data on cost and performance comparisons for different models and configurations up front, simplifying the transition to cloud computing on the Azure Cloud infrastructure.” - Peter Blaser, Vice President of Operations, CPFD

“At ENGYS we are delighted to have participated in this Cook Book certification programme with the help of Microsoft and CAPGemini. We are very excited about the prospects of offering Azure’s on-demand HPC resources to our customers to exploit the unlimited scalability of our open-source based CFD tools HELYX and ELEMENTS. The concept of on-demand cloud computing is perfectly aligned with our open license approach to release the full potential of our simulation software.” - Francisco J. Campos, Director of Operations, ENGYS

“KeyShot is pleased to participate in this program with Microsoft Azure and Capgemini to feature and validate our real-time 3D rendering software and deployment instructions on Azure infrastructure. This, along with the NVIDIA GPU accelerated instances on Azure, will allow our customers to create realistic product visuals, communicate ideas easier, explore concepts sooner, and deliver jaw-dropping visuals faster!” - Derek Cicero, Vice President of Products & Strategy, KeyShot

“We invite our HPC AI ISV partners to participate in this program – it represents a commitment to our partners to grow their business by making it easy to onboard our shared customers to the cloud” - Kurt Niebuhr, Principal Program Manager, Azure Core

“Metariver Technology is pleased to be part of this program jointly with Microsoft and Capgemini. In particular, it is because of the excellent Azure infrastructure environment that we were able to complete the validation of 3 softwares from our "samadii" series in a very short time. "samadii" is a product that performs particle-based simulations using GPU parallel computing technology, which uses a lot of computation resources and requires high-density calculations. It has been confirmed that such computational performance is smoothly possible on Azure cloud. We expect that many customers in the global market that require various particle-based computations will have more opportunities to check and meet our products.” - Cheol-O Ahn, CTO, Metariver Technology Co., Ltd.

“Thanks to the great efforts by Microsoft to certify a multitude of NVIDIA GPU-accelerated Engineering ISV applications, including Siemens NX and Altair UltraFluidX amongst others, customers can now deploy CAD and CAE workflows on Azure with confidence.” - Sean Young, Director Business Development, NVIDIA

“Microsoft Azure with Capgemini have connected NVIDIA A100 GPU's to Remcom's industry leading time-domain EM solution. Using this infrastructure, Remcom is able to further expand and run some of the largest full-wave electromagnetic time-domain simulations ever. Together with this collaboration, engineers can easily prototype devices and enable future technologies” - Tarun Chawla, Director Business Development, Remcom

How to get started?

If you use high performance computing applications for your product design:
  • Do you want to evaluate migration to Azure HPC?
  • Do you want to validate a specific workload on Azure HPC?
If you develop software that requires high performance computation:
  • Do you want a cookbook for your application to feature here?
  • Do you want to promote applications capability on cloud infrastructure and showcase performance?
 
  • Reach out and get in contact with Azure Compute or email Capgemini to have your application prioritized
  • Provide our partner, Capgemini, with a software license and model to complete the performance evaluation needed
  • Assign a technical resource to assist Capgemini and Azure in optimization efforts
